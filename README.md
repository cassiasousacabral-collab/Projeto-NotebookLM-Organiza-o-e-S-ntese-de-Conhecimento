# Projeto-NotebookLM-Organiza-o-e-S-ntese-de-Conhecimento
Projeto desenvolvido durante curso na DIO com o objetivo de aplicar o uso do NotebookLM na organização, análise e síntese de conteúdos. O projeto demonstra como a ferramenta pode apoiar processos de estudo, pesquisa e geração de insights a partir de materiais estruturados.
